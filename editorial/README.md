# Editorial — Sistema Contenuti IG

## Come usare questi file

### 1. Per pianificare una nuova settimana
Apri il brief del progetto → vai alla sezione **11. PROMPT GENERAZIONE CONTENUTI** → copia il blocco → apri una **nuova chat Claude** → incollalo con `SETTIMANA [N]` e `OBIETTIVO` compilati → Claude genera tutti i contenuti pronti.

### 2. Per aggiornare la strategia
Modifica direttamente il file `.md` del progetto. Ogni sezione è autonoma.

### 3. Per un nuovo progetto
Copia `TEMPLATE.md` → rinomina → compila tutte le sezioni.

## Struttura di ogni brief

1. **Prodotto** — USP, CTA, dove si converte
2. **Profilo IG + Linktree** — bio ottimizzata, link tree
3. **Target & Persona** — chi è, pain points, desideri, comportamento IG
4. **Strategia Lead & Conversione** — funnel in 3 step, lead magnet, CTA
5. **Pillar Content** — 5 temi editoriali ricorrenti
6. **Identità Visiva** — palette, font, mood, stile grafico
7. **Struttura Settimana Tipo** — tabella giorno/formato/CTA
8. **Settimana 1** — contenuti pronti (post completi + story sequence)
9. **Programmazione** — tool, workflow, frequenza
10. **KPI & Obiettivi** — baseline, target 30/90gg
11. **Prompt Generazione** — copia/incolla per nuove settimane

---

## Workflow settimanale consigliato

```
Domenica sera:
1. Apri il brief del progetto
2. Copia il Prompt (#11) in una nuova chat
3. Aggiungi: SETTIMANA [N] + OBIETTIVO SETTIMANA
4. Claude genera i contenuti completi
5. Review rapida (5 min)
6. Grafica su Canva (palette già definita)
7. Carica su Meta Business Suite / Later
8. Programma per la settimana
```
